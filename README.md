# Projeto-UniCheck
📖 Descrição

UniCheck é uma plataforma de educação voltada para faculdades, criada para facilitar a entrada de calouros que muitas vezes chegam sem informações relevantes sobre o ambiente acadêmico. A plataforma foi desenvolvida para atender tanto estudantes quanto instituições de ensino, promovendo uma experiência mais organizada e informativa no início da vida universitária.

⚙️ Tecnologias Utilizadas

HTML

CSS

JavaScript

🚀 Como Rodar o Projeto

Baixe ou clone o projeto.

Extraia os arquivos compactados (se houver).

Abra o projeto no VS Code ou em outro editor de sua preferência.

Abra o arquivo index.html no navegador para visualizar a aplicação.
